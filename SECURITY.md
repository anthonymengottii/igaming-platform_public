# Política de Segurança

## Reportando Vulnerabilidades

Se você encontrar alguma vulnerabilidade de segurança relacionada a este projeto, por favor:

1. **NÃO** abra uma issue pública no GitHub
2. Entre em contato diretamente através do email ou GitHub
3. Forneça detalhes sobre a vulnerabilidade encontrada

## Nota Importante

Este repositório contém apenas a **documentação** do projeto. O código-fonte é mantido em repositório privado.

Se você encontrar vulnerabilidades relacionadas à documentação ou informações expostas aqui, por favor, reporte-as seguindo o processo acima.

## Responsabilidade de Segurança

Como este é um repositório de documentação apenas, não há código executável ou credenciais expostas neste repositório. Todas as informações sensíveis são mantidas no repositório privado do código-fonte.
